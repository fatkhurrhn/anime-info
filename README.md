# AnimeInfo - Anime Information Website

AnimeInfo is a comprehensive anime information website that provides detailed information about anime including latest episodes, genres, studios, seasons, and search functionality. No streaming, just pure anime information!

## ✨ Features

- **Latest Anime** - View the latest anime releases with episode information
- **Anime Details** - Complete anime information including synopsis, episodes, genres, studios, and ratings
- **Search Anime** - Search anime by title with auto-suggestions and pagination
- **Genres** - Browse anime by 77+ genres with detailed genre pages
- **Studios** - Explore anime by 295+ animation studios
- **Seasons** - Filter anime by seasonal releases (Winter, Spring, Summer, Fall)
- **Dark Mode** - Beautiful dark theme optimized for mobile devices
- **PWA Support** - Install as native app on mobile and desktop
- **Responsive Design** - Mobile-first design with 3-column grid layout

## 📦 Tech Stack

- **Frontend**: React 18
- **Routing**: React Router DOM v6
- **Styling**: Tailwind CSS
- **Icons**: Remix Icon
- **API**: Custom anime API
- **PWA**: Vite PWA Plugin / Next PWA
